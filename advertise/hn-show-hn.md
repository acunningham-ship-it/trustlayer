## Body

```
I built TrustLayer after getting burned by AI hallucinations one too many times.

The problem: I use 4-5 different AI tools and have no way to know when any of them
are making things up, how much I'm actually spending, or which model is actually best
for *my* work.

TrustLayer wraps around any AI you already use and adds:
- Trust scores (0-100) on every response with flagged claims
- Hallucination detection and source attribution
- Real-time cost tracking across all your AI providers
- Model comparison for your actual tasks (not synthetic benchmarks)
- Local knowledge base — index your docs and use them offline with Ollama
- No-code workflow builder

It auto-detects Ollama if you have it running. Works with any OpenAI-compatible API.

Install: pip install trustlayer-ai
Start: trustlayer server

It's fully open source. Would love feedback from this community.

GitHub: https://github.com/acunningham-ship-it/trustlayer
```
