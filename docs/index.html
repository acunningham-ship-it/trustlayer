How it works

Simple to start. Powerful over time.

TrustLayer sits between you and every AI you use. You don't change your workflow — you just gain visibility and trust.

1

Connect your AIs

Ollama is auto-detected. Add API keys for Claude, GPT-4, or Gemini via the Settings page or environment variables. No account required for TrustLayer itself.

2

Every output gets verified

As you use AI, TrustLayer silently scores each response. Trust scores, hallucination flags, and source attribution appear inline — without slowing you down.

3

It learns your patterns

TrustLayer remembers your sessions, preferences, and writing style. Over time, it surfaces better suggestions, cheaper alternatives, and personalized tips.

Quick Start

Simple by design

Works with everything you already have. Zero config. One command to start.

# Install TrustLayer
pip install trustlayer

# Start the server + web UI
trustlayer server
# → Auto-detects Ollama
# → Opens http://localhost:8000

# Add your API keys (optional)
export ANTHROPIC_API_KEY=sk-ant-...
export OPENAI_API_KEY=sk-...
# Verify any AI output
trustlayer verify "The earth is 4.5 billion years old."
# → Trust Score: 94/100 (HIGH) — No concerns

# Ask any connected AI
trustlayer ask "Summarize this codebase" --provider ollama --model llama3.2

# Compare responses from multiple models
trustlayer compare "Write unit tests for this function"

# Check your spending across all providers
trustlayer costs

# Detect available AI tools on your machine
trustlayer detect
# Verify content via REST API
curl -X POST http://localhost:8000/api/verify \
  -H "Content-Type: application/json" \
  -d '{"content": "Your AI output here"}'

# Response:
{
  "trust_score": 87,
  "trust_label": "high",
  "summary": "87% trusted. 0 concerns flagged.",
  "issues": []
}

# Compare models on the same prompt
curl -X POST http://localhost:8000/api/compare \
  -d '{"prompt": "Explain quantum entanglement"}'
import requests

# Verify any AI output
response = requests.post("http://localhost:8000/api/verify", json={
    "content": "AI output to verify"
})
result = response.json()
print(f"Trust: {result['trust_score']}/100 — {result['trust_label']}")

# Compare providers side by side
comparison = requests.post("http://localhost:8000/api/compare", json={
    "prompt": "Your task here",
    "providers": ["ollama", "anthropic", "openai"]
}).json()
